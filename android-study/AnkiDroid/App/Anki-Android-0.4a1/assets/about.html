Ankidroid:
Flashcards for Android

Ankidroid can read .anki files produced by the desktop Anki software.

Warning: Ankidroid modifies .anki files in a way that is not yet compatible with the Anki desktop application. Do not copy .anki files from the SD card back to your computer.

You can report bugs and request new features or improvements here, discuss on the forum or view the official wiki.

Ankidroid is open source software, and everyone is encouraged to get the source from here or here and build on it :-)
